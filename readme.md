# Install Lamp server and secure it on debian 7

## how to use it
on a fresh install
1 install git
```
sudo apt-get git
```

2 clone the repo
```
git clone https://github.com/bachy/debian-lamp.git
```

3 run the script as root
```
su
cd debian-lamp
chmod a+x install-debian-server.sh
./install-debian-server.sh

```
